# 手机版 - 需求文档

## 目标
将心情音乐盒做成手机 APK，独立运行，不依赖电脑。

## 与网页版的区别

| 对比项 | 网页版 | 手机版 |
|--------|--------|--------|
| 运行环境 | 电脑浏览器 | 手机 APK |
| API 来源 | 本地 localhost:3000 | Railway 公网服务器 |
| 启动方式 | 双击 start.bat | 点击桌面图标 |
| 网络要求 | 无需联网 | 需要联网（调用云API） |
| 部署方式 | 本地文件 | 打包成 APK 安装 |

## 核心功能（与网页版一致）
- [x] 游客模式 / 自建账号登录
- [x] AI 聊天搜歌（DeepSeek）
- [x] 歌曲搜索播放（网易云 Railway API）
- [x] 播放控制（播放/暂停/上下首/进度条）
- [x] 音量调节
- [x] 歌词同步显示
- [x] 播放列表（独立 Tab）
- [x] 收藏管理
- [x] 个人中心
- [x] 记忆播放（恢复上次播放）
- [ ] 语音输入（后续）
- [ ] 心情匹配推荐（后续）

## 技术方案
- PWA 打包 APK：使用 PWABuilder 或 Bubblewrap
- API：DeepSeek + Railway 网易云 API
- 全前端，无后端
